# Task Manager API

A simple full-stack Task Manager application built using:

- FastAPI
- SQLite
- JWT Authentication
- HTML/CSS/JavaScript

## Features

- User Registration
- User Login
- JWT Authentication
- Create Tasks
- View Tasks
- Complete Tasks
- Delete Tasks
- Protected Routes

## Backend Setup

```bash
cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
```

## Frontend Setup

Open:

```text
frontend/index.html
```

using Live Server.
